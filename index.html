제 1조 #기본규정

1. 모든 신고는 총관리자가 판단 하에 처벌이 결정됩니다.
2. 모든 유저는 규칙 위반자를 신고할 의무가 있으며 묵인, 방관 할 시 처벌당할 수 있습니다.
3. 서버 내 모든 채팅은 저장되며 관리진은 채팅을 볼 수 있으며 증거자료가 필요할 때 참고될 수 있습니다.
(이 외의 목적으로 사용하지 않습니다)

해당 규칙은 MSG 서버에서 작성되었으며,
내용을 허가 없이 복사 또는 수정하는 경우 처벌이 진행됩니다.

! 본 법전 게시글은 2021년 12월 10일부터 적용되며,
본 게시글에 기재되지 않은 사항이라도 서버대표 판단하에 처벌당할 수 있습니다.

! 처벌 방식 또한 관리자 판단하에 바뀔 수 있으며, 처벌 방식은 다음과 같습니다.
( 인벤토리 초기화 / 벌금 / 랭크 초기화 / 마을 초기화 / 강제 철거 / 경고 / 밴 / IP밴 )
