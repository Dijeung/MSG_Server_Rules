제 1조 #기본규정

1. 모든 신고는 총관리자가 판단 하에 처벌이 결정됩니다.
2. 모든 유저는 규칙 위반자를 신고할 의무가 있으며 묵인, 방관 할 시 처벌당할 수 있습니다.
3. 서버 내 모든 채팅은 저장되며 관리진은 채팅을 볼 수 있으며 증거자료가 필요할 때 참고될 수 있습니다.
   (이 외의 목적으로 사용하지 않습니다)

해당 규칙은 MSG 서버에서 작성되었으며,
내용을 허가 없이 복사 또는 수정하는 경우 처벌당할 수 있습니다.

! 본 법전 게시글은 2021년 12월 10일부터 적용되며,
본 게시글에 기재되지 않은 사항이라도 서버대표 판단하에 처벌당할 수 있습니다.

! 처벌 방식 또한 관리자 판단하에 바뀔 수 있으며, 처벌 방식은 다음과 같습니다.
( 인벤토리 초기화 / 벌금 / 랭크 초기화 / 마을 초기화 / 강제 철거 / 경고 / 밴 / IP밴 )

! 반성문을 통해 밴이 풀리게될때는 밴 사유에 따라 다음과 같은 처벌이 이루어집니다.
( 인벤토리 초기화 / 벌금 / 랭크 초기화 / 마을초기화 / 엔더상자 초기화 / 창고 초기화 )

   - 밴 경력이 있을경우 경고 1회만 받더라도 밴처리 됩니다.




제 2조 #채팅 규정 💬

1. 이벤트
: 채팅을 꺼두어 통보를 못받은 사항에 대한 피해보상은 해드리지 않습니다.

2. 단타채팅이란?
: 채팅 질서에 혼란을 줄 정도로 단타 채팅을 지속하는 행위입니다.

   - 증거 스크린샷 내에 고의성 및 악의성 또는 3자가 보아도 
    채팅 혼란을 야기할 수 있다고 보여질 때 신고가 가능합니다.

   - 단타 채팅 자제 요청을 무시할때 신고가 가능합니다.

3. '공지북' 아이템 주의사항!
: 공지북을 이용하지 않고 전채채팅으로 홍보할 경우 처벌대상입니다.
  ( 신입분들은 실수로 하셨을경우 제외되며, 연속적일 경우 처벌대상입니다 )

4. 욕설 🤬
: 서버 내에서 욕설, 어그로, 성적 용어, 동성애 언급,
  패드립, 채팅 유도 등의 행위를 말합니다.

   - 채팅 상황에 따라 지속적인 사투리 또한 처벌 대상에 포함될 수 있습니다. ( 일베 어그로 등 )

   - 아이템에 모루를 사용하여 변경한 부분 또한 포함되며, 
     인게임 닉네임 ( 마인크래프트 닉네임 ) 또한 포함됩니다.

   - 수위가 낮은 욕설은 신고글이 올라오면 처리됩니다.

   - 특정 단어가 욕설로 등록된 단어가 아니더라도 
     관리자 판단하에 처벌될 수 있습니다.

! MSG 서버 관리진은 모든 지역과 방언을 존중하며, 차별 의도는 전혀 없음을 고지합니다.

5. 접그로
: 서버를 더 이상 플레이 하지 않으려는 
  의사를 공개적으로 또는 지속적으로 표출하는 행위

6. 비하
: 제 3자가 보이게도 확실히 비하로 보이는 경우 처벌 대상입니다.
  피해자가 명확히 존재할 때 최대한 요청에 기반하여 처리됩니다.

7. 타 게임 유도
: 본 서버에서는 타 게임 발언은 처벌 대상이 아니지만, 
  타 게임을 같이하자고 유도하는 행위는 금지하고 있습니다.
  ( 단, 지나친 발언 수위는 재제될 수 있습니다 )

   - 간접적으로 또는 특정 그룹에 소속되기 위한 조건으로도 금지되며, 
     게임 플레이와 관련된 유도를 금합니다.

8. 지속적인 구걸 행위
: 지속적으로 구걸을 하는 행위를 말합니다. ( OP 구걸 포함 )

9. 친목
: 공개된 채팅에서 지속적으로친목을 하는 행위를 말합니다.

   - 게임 특성상 친목 행위 자체를 제한하지 않습니다.
    
   - 가벼운 반말과 인사말 은 허용되나, 
     관리자 기준 보기 거북할 정도의 도가 지나친 채팅을 금지합니다.
    
   - 이는 처리 과정에서 일어나는 의견 충돌을 방지하기 위함이며, 
     모든 관리진과 논의 후 공정히 처리합니다.
    
   - 모든 유저는 위 상황에 대해 자제할것을 요청할 수 있으며,
     이를 무시하고 계속해서 감행한다면 신고 권한이 주어집니다.
    
10. 서버 비하
: 서버를 비하하는 행위

11. 무고죄
: 정확한 증거 스크린샷 없이 또는 타인을 신고하기 위해 조작된 상황을 만들어 신고하는 행위

12. 정치 발언
: 타인이 볼때 과도할 정도로 수위 높은 정치 발언을 하는 경우를 말합니다.

13. 타 서버 언급
: 타 서버 발언은 금지입니다.

   - 서버 내에서 다른 서버를 홍보하는 행위를 말하며,
     직접적인 서버 이름을 언급하지 않아도 서버로 유도하는 행위를 금지합니다.
    
   - 고의적으로 타 서버 이름을 반복하여 언급하는 행위 또한 처벌 대상입니다.

14. 분쟁 / 부적절한 언행
: 서버 내부에서 유저간 갈등으로 인하여 소란을 피우는 행위

15. 본인 요청
: 본인에게 밴이나 채팅 금지를 요구하는 경우

16. 링크
: 채팅 내 외부 링크를 작성하여 올리는 행위를 말합니다.

   - 서버 카페 링크나 서버 디스코드 등 서버 관련 링크를 올리는것은
     상관 없으나, 타 서버 카페 링크 또는 광고성 링크는 처벌 대상입니다.

17. 분쟁
: 경고 / 도배 / 밴 이런 단어로 더 큰 분쟁을 일으킬 수 있으며, 관리진 / 스태프 이외 유저들은 사용금지

18. Join / Quit
: 3초 무적을 목적으로 나갔다 들어오는 행위를 연속적으로 하여,
  채팅을 도배시키는 행위는 처벌대상입니다.

   - 위에 적힌 목적만 해당되는게 아닙니다.
     만약 서버문제라면 관리자에게 연락주시기 바랍니다.




제 3조 #유저 간 커뮤니케이션 👫

1. 유저에게 방해가 되는 플레이
: 무단 침입 ( 자신의 영역임이 증명되거나[관리자 판단], 마을 보호가 되있을 경우 )

   - 무단 침입의 경우 나가라는 말을 하신 후 무시하였을때 처벌 가능합니다.

   - 지옥 월드에서 포탈을 막는 행위 또한 처벌 대상이며, 작물 서리 등도 포함됩니다.

2. 무단 PvP
: 유저간 동의 없이 상대를 치거나 죽인것을 말합니다.

   - 만일 상대에게 죽었을 경우 즉시 관리자나 스태프에게 전달 바라며, 
     똑같이 죽일 경우 신고자 또한 처벌이 가해질 수 있습니다.

   - 무단 PVP 판단은 로그를 통해 판별됩니다.

   ※ 약탈월드에서는 해당되지 않습니다.

3. 스폰
: 서버 스폰(로비) 내에서 불화살을 발사하거나,
  유저에게 불편을 주는 행위 등은 강력히 처벌됩니다.

4. 소유물 분실
: 타 유저의 아이템을 빌리거나 거래 도중 분실하였을 때
  혹은 접속을 종료하였을 때 피해자는 손해배상을 요구할 수 있습니다.

5. 스킨 사용
: 특정 종교, 단체 등을 연상시키는 스킨 사용이 금지되며,
  성적 표현이 묘사된 스킨 또한 사용이 금지됩니다.

6. 유저간 대출 행위
: 대출에 대한 부분은 서버에서 관여하지 않으며 제한하지 않습니다.

   - 다만 신뢰할수 있는 플레이어가 아닌 플레이어에게는 아이템 또는 포인트를 빌려주는 행위를 절대 권장하지 않으며, 
     빌려간 플레이어가 차단 또는 서버 접속을 하지 않아 받지 못한 손해는 플레이어의 계좌 또는 인벤토리에 
     아이템 또는 포인트가 남아있는 경우 빼드릴수는 있으나 이 외의 경우 서버측에서 보상해주지 않습니다.

7. 성희롱
: 특정 유저에게 성적 상징 또는 문구나 
  단어를 보내거나 보여주었을 경우를 말합니다.

8. 실수
: 타인이 흘린 아이템을 주웠고 그 아이템의 주인이 아이템의 복구를 요청할 경우 
  아이템의 소유권은 본 주인에게 있으며 돌려줘야한다.

9. 현금거래
: 유저간에 실제 현금거래는 처벌대상 입니다.




제 4조 #핵 / 모드 규정 💾

1. 매크로
: 매크로 검사중 귓속말에 응답이 없을 경우 매크로로 간주하여
  처벌될 수 있으며, 어떠한 이유가 있어도 철회되지 않습니다.

2. 모드
: 서버에서 금지하는 모드를 사용하는것을 말합니다.

   - 차단 모드는 플레이에 지장을 주거나 더 편리하게 동작을 실행하는 모드를 말합니다.
     ( 엑스레이 , 핵 , 스마트 무빙 , 매크로 , 오토 마우스 , 리플레이 모드등 )
     
   - 위 목록에 해당하는 모드가 아니더라도 서버 플레이에 영향을 주거나 
     비정상적인 움직임을 가능하게 하는모드 처벌대상 입니다.

3. 낚시
: 낚시 회로 및 자동 낚시 ( 매크로 ) 를 금지하고 있습니다.




제 5조 #마을 & 땅 규정 🏡

1. 미접속 마을
: 3주 (30일) 이상 미접속 유저의 경우 건축물을 포함한 
  마을보호가 제거될 수 있으며, 모든건물을 제거합니다.
  ( 단, 마을에 다른유저가 접속중일 경우는 해당되지 않습니다. )

   - 장기간 미완성으로 방치된 건축물은 철거 작업이 이루어 질 수 있습니다.
     ( 일반 밴 유저는 3일 이내 반성문 없을 경우 )

2. 테러 / 복구
: 테러 복구 대상은 보호구역에 한정하여 복구되며, 대형 용암 / 물테러도 해당됩니다.

   - 테러 복구 대상에 도트가 포함될경우 도트는 복구하지 못합니다.
     ( 채워진 지도류 복구 불가 )

   - 서버로부터 개인이 피해를 받은 경우 정확한 증거 (영상), 
     스크린샷이 요구되며 없을 경우 복구되지 않습니다.

   - 타인의 건축물을 파괴하거나 상자 등을 약탈하는 경우를 말합니다.

   - 타인의 건축물을 파괴하거나 상자 등을 약탈하는 경우를 말합니다.
     ( 마을월드는 해당되지 않습니다 )

3. 마을 규칙 위반
: 관리자는 마을에 관여하지 않으며,
  마을 규칙을 위반하여도 서버 법에 적용되지 않습니다.

4. 주변 건축
: 외부인이 마을 또는 보호구역 근처에 
  건축 또는 훼손을 하는 행위를 말합니다. ( 100칸 이내 )

   - 단, 마을 생성일 또는 구역 생성일보다 
     먼저 지어진 건축물은 적용되지 않습니다.

   - 악의적으로 타 플레이어의 건축물 방해한다고 
     생각될 때 또는 서버 장기간 미접속 상태일 때는 처리 가능합니다. 

   - 이 경우 집이 옮겨지거나 철거될 수 있습니다.

   - 구역 주변 수직탑 등은 무단 침입 시도로 간주될 수 있습니다.

   - 마을이 생성되어있지만, 고의적으로 땅을 넓히지 않는다면 테러를 당했을 시 
     가해자만 처벌을 받는것이 아닌, 피해자도 처벌을 받을 수 있습니다.

5. 무단친입
: 외부인이 자신마을에 무단친입할 경우 처벌대상입니다.

6. 타 마을 셋홈
: 외부인이 자신마을이 아닌 다른마을에 셋홈할 경우 처벌대상입니다.




제 6조 #몬스터 / 회로 🧟‍♂️

1. 회로 규정
: 무한 회로 및 용도 없는 회로 등 
  서버에 지장이 생길 정도로 거대한 회로는 금지됩니다.

   - 자동문 / 도박장 정도의 간단한 회로는 허영되나, 
     비밀번호 회로 , 자동 농사 회로등은 처벌대상입니다.

2. 자동화 농장
: 본 서버에서는 어떤 방법이든 자동화로 진행되는 농사를 금지합니다.

   - 다량의 깔때기를 이용한 기계도 똑같이 적용되며, 
     물 양동이를 이용한 재배는 제외됩니다.

3. 생성기
: 조약돌을 제외한 모든  생성기 ( 눈사람을 이용한 눈 생성기 등 )
  제작을 금지하고 있습니다.

4. 엔티티 규정
: 기본적으로 몹 타워 제작은 제한됩니다.

   - 주변을 어둡게 만들어 몬스터를 스폰되게 하는 던전 제작도 제한됩니다.
   
   - 자연적으로 스폰 된 몬스터들이 공격할 수 없는 구조물이나 물 또는 기타 아이템, 
     AI 특성을 이용하여 한곳으로 모으는 행위는 금지됩니다.

   - 스폰 된 몬스터들이 플레이어의 공격 이외의 
     데미지를 입도록 만든 시설 또한 처벌 대상입니다.

   - 몬스터 스포너가 작동하는 범위 ( 몬스터 스포너가 활성화 되는 범위 ) 
     내에서 몬스터들에게 맞지 않는 구조물은 처벌 대상입니다.

   - 철골렘,  눈사람, 돼지 좀비, 슬라임 등을 
     이용한 일명 철농장, 금농장 등을 제한합니다.




제 7조 #시스템 💻

1. 계정 규정
: 계정 해킹 피해는 서버에서 책임지지 않습니다.

   - 마인크래프트 닉네임을 변경하시기 전 주의사항을 확인 후 변경해주시기 바라며, 
     개인 실수는 서버에서 책임지지 않습니다.

   - 서버에서 부계정 사용이 금지됩니다.

   - 같은 장소 ( 아이피가 동일할 때 ) 에서 플레이 하기 전 
     반드시 관리자에게 연락을 해주시기 바랍니다.
     
   - 실제 현금 거래를 하면서 대리접속을 하는 행위를 금지하고 있습니다.
     현금 거래가 아니더라도 대리접속 자체가 금지되니 참고부탁드립니다.

   - 대리접속이 적발될 경우 대리접속을 하시는분이 처벌대상 되실 수 있습니다.

2. 서버 시스템 규정
: 본 서버는 엔티티가 10분에 한번씩 초기화 되며, 이에 따른 피해보상은 없습니다.

   - 본 서버에서는 주민과의 거래가 불가능합니다.
   
   - 1인당 최대 허용 엔티티 수는 30마리로 제한됩니다. ( 늑대, 양, 돼지, 소 등 모두 포함 )
     어길 경우 강제 철거 또는 경고처리 됩니다. 규모에 따른 처벌 UP

3. 사기 규정
: 약속된 계약을 어기는 행위.
  마을 내부 일은 관여하지 않으며, 돈을 지급하지 않는 행위

4. 사칭
: 특정 플레이어를 사칭하는 행위

   - 닉네임이 비슷하다는 이유로는 처벌할 수 없으며,
     이로 인하여 생긴 피해 또는 행세하는 행위는 처벌이 가능합니다.

   - 특정 그룹 사칭 또한 적용됩니다.

5. 무단 복제
: 타인의 도트나 다른 창작물을 무단으로 복제하는 경우를 말합니다.

   - 무단 판매 또한 똑같이 적용됩니다.

6. 아이템 이름
: 관리진 & 스태프 이름이 언급된 아이템을 제작 및 판매시 경고대상 입니다.

   - 아이템의 이름을 변경하여 유저를 속이려 하는 행위 등은
     어그로로 판단하여 처벌합니다.

   - 아이템 이름은 최대 8자를 넘어서는 안됩니다.




제 8조 #기타 📔

1. 처벌 규정
: 반성문은 영구 차단 유저는 작성이 불가하며, 1회 작성 가능합니다.

2. 신고 규정
: 차단 기록이 있는 유저는 1회 경고 부여 시 바로 차단됩니다.
  사건처리에 대한 이의 제기는 7일 내로 진행 부탁드립니다, 
  이후 관련 문의는 받지 않습니다.

3. 타 서버
: 타 서버에서 MSG 서버 이미지를 훼손하는 행위
  ( 타 서버에서 MSG 서버를 홍보하는 행위 등 )
   
   - 타 플렛폼 ( 디스코드, 트위치 등 ) 해당됩니다.

4. 수직굴
: 수직굴(Y좌표)은 처벌대상이지만, 
  블럭으로 입구를 막거나 물을 설치 할 경우 처벌 대상에서 벗어난다.

5. 답변
: 늦은시간에 계속 답변을 해달라는 행위

6. 닉네임
: 부적절한 닉네임 사용금지 
  ex ) 정치 / 19금 등
